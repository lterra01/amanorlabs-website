Khomp Agent

Khomp Agent is an evidence-based assistant for product documentation, support, sustaining and project teams. It starts clean and allows authorized administrators to upload documents directly into a private Datalake.

This public route is reserved for the Khomp Agent application. The app requires a backend runtime for uploads, audit logs and secure OpenAI access, so GitHub Pages alone cannot execute the full service.

The runtime package is prepared for deployment under /khompagent/. Once the backend service is attached to this route, administrators can load the Khomp product documents directly in the application.
